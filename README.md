# 🌱 Growing Code – Python Fundamentals

This project is part of the **42 learning journey**.  
It focuses on **Python fundamentals** through simple exercises inspired by a community garden.

**Project Goals:**  
- Understand Python syntax  
- Functions  
- User input  
- Control flow  
- Iteration & Recursion  
- Basic problem solving  

Each exercise is a small, practical task.

---

## 📚 Exercises

**Exercise 0 – Hello Garden**  
Print a welcome message for the garden community.  
**Concepts used:** Functions, `print()`

**Exercise 1 – Garden Name**  
Ask the user for the garden name and display a summary.  
**Concepts used:** `input()`, Variables, Output formatting

**Exercise 2 – Plot Area**  
Calculate the area of a garden plot using length and width.  
**Concepts used:** `int()`, Arithmetic operations

**Exercise 3 – Harvest Total**  
Add the harvest amounts from multiple days.  
**Concepts used:** Variables, Addition

**Exercise 4 – Plant Age Check**  
Check if the plant is ready to harvest.  
**Concepts used:** `if / else`, Conditions

**Exercise 5 – Water Reminder**  
Check if plants need watering.  
**Concepts used:** Conditional logic

**Exercise 6 – Count to Harvest**  
Two implementations:  
- Iterative (using loops)  
- Recursive (function calling itself)  
**Concepts used:** `range()`, Loops, Recursion

**Exercise 7 – Seed Inventory**  
Display seed inventory using type hints.  
**Concepts used:** Type annotations, Function parameters, String formatting
